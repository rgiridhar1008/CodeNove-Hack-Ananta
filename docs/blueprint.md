# **App Name**: Civix

## Core Features:

- Issue Reporting: Allow users to report civic issues with geo-tagging.
- Transparent Tracking: Enable users to track the status and resolution of reported issues.
- Community Voting: Implement a voting system for community matters and polls.
- Data-Driven Insights: Provide statistical insights on reported issues and resolution times.
- Feedback Collection: Gather user feedback via emoji ratings and feedback forms.
- Issue Prioritization Tool: AI Tool that analyzes feedback data and helps to suggest the correct level of issue prioritization based on public sentiment and critical factors, for more effective allocation of resources.

## Style Guidelines:

- Primary color: Blue (#2563EB) for a professional and trustworthy look.
- Accent color: Green (#22C55E) for positive actions and highlights.
- Background color: Light gray (#F9FAFB) to ensure a clean and accessible layout.
- Font: 'Inter', a sans-serif font known for its readability and modern design, is used for both headlines and body text.
- Lucide icons are used for a consistent and clean interface.
- Mobile-first, responsive layout with soft shadows and rounded cards for a modern look.
- Smooth hover and transition animations to enhance user interaction.
